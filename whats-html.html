HTMLについて

まずは基本から

HTMLとは

HTMLとは、WEBページを構成するための言語です

以下のようなタグに囲まれたテキストで表現をします

<!DOCTYPE html>
<html>
    <head>
        <title>タイトル</title>
    </head>
    <body>
        <p>表示される内容</p>
    </body>
</html>

実際に書いてみよう

ここからtemplate.htmlをダウンロードしましょう

ダウンロードしたファイルをVisual Studio Codeで開いてください

template.htmlファイル内の<section>タグの中にコードを記述していきます

まずは、<div class='box'>内のh1、pタグのテキストを変更してみましょう

<h1>タグは見出し、<p>タグは段落の意味になります

次に以下のようなタグを追加します

<button class='btn'>ボタン</button>

これでボタンを実装することができます

次に画像を表示させます

<img src='http://itnav.jp/wp/wp-content/themes/itonabu/img/logo_top.png' alt='画像の説明'>

画像の表示は<img src='画像のパス' alt='画像の説明'>このように記述します

次にリストを表示させます

リストを表示させるには、<ul>タグを使用します

<ul>タグは順序のないリストを表示する際に使用します

リストの各項目は<li>タグを用いて記述します

また順序のあるリスト表示には<ol>タグを使用します

以下のコードをコピーして実行してみましょう

<h3>好きな動物</h3>
<ul>
    <li>白くま</li>
    <li>犬</li>
    <li>猫</li>
</ul>

<h3>カレーの作り方</h3>
<ol>
    <li>肉と野菜を炒める</li>
    <li>水を入れて中火で煮込む</li>
    <li>ルーを入れて弱火で煮込む</li>
</ol>

そして最後にもう１つ、リンクタグを使いたいと思います

リンクタグは以下のように書きます

<a href='http://itnav.jp/' target='_blank'>イトナブサイトへ</a>

<a>タグはアンカーの略で、hrefで指定したurlのサイトへ移動します

そしてtarget='_blank'で新規のタブでウェブサイトを開くことができます

こちらがないと現在のページでリンク先が表示されます

さらにこちらからHTMLのタグ一覧を確認できます

スタイルをつけてみよう

HTMLのみでは書くことはできても見た目があまり良くありません

スタイルシートを書くことでよりかっこよいページをつくることができます

以下のようなセレクタ(対象)に対して区切られた範囲の中で、スタイルを定義します

.btn {
    min-width: 7.5em;
    background-color: skyblue;
    cursor: pointer;
    padding-top: 0.6em;
    padding-bottom: 0.6em;
    border: 1px solid black;
    border-radius: 0.5em;
    outline: none;
    appearance: none;
    transition: all 0.25s ease;
}

.btn:hover {
    opacity: 0.8;
}

.btn:active {
    opacity: 1.0;
    transform: translate(0.1rem, 0.1rem);
}

HTMLの中でスタイルを定義するには<head>タグ内でスタイルを定義します

template.htmlでは既に<style>タグが定義されているのでその中に書くことになります

<style>タグの中に上記のスタイルシートをコピペしてみましょう

ボタンの見た目が変わったはずです

<button class='btn'>ボタン</button>のbtnとスタイルシートの.btnが対応しています

classを対応させる場合はドット(.)を名前の前に、idを対応させる場合はシャープ(#)を名前の前につけてスタイルシートを定義します

Bulmaを使ってみよう

前章のように直接自分でスタイルシートを書くこともできますが、非常に手間が多く、ボタン一つの機能を定義するだけでも15行ほど書く必要がありました。全て自分でスタイルを定義するととんでもない労力になります

そこでCSSフレームワークというものが登場しました。今回は新興フレームワークのBulmaを使います

Bulmaが提供する主な機能は以下のようなものがあります

  • 主要なパーツのデフォルトスタイル
  • warningやdangerなどのコンテキストカラーの定義
  • パーツを作るためのスタイル群(このページのタイトル部分のような)
  • モバイル・デスクトップ両対応ページ

Bulmaの機能を利用するには⁢head>タグに以下の読み込み文を追加します

<!-- ページのスケールを設定 -->
<meta name='viewport' content='width=device-width, initial-scale=1'>

<!-- icon -->
<link rel='stylesheet' href='https://cdn.materialdesignicons.com/2.6.95/css/materialdesignicons.min.css'>
<script defer src='https://use.fontawesome.com/releases/v5.1.0/js/all.js'></script>

<!-- bulma -->
<link rel='stylesheet' href='https://cdnjs.cloudflare.com/ajax/libs/bulma/0.7.1/css/bulma.min.css'>

Bulmaの機能の利用例は以下のようなものがあります

<button class='button'>ボタン</button>
<i class='icon fa is-large fa-home'></i>

スクリプトを書いてみよう

WEBページに様々な処理を実装するために、スクリプト(プログラム)と言うものを書きます

今回はボタン要素をクリックして、ダイアログを表示させてみます

下記のようにダイアログを表示させる関数(タスクや値計算を実行する文の集まり)を書いてみましょう

function sayHello() {
    alert('Hello World');
}

関数を定義するにはまず、functionと記述し関数を定義することを明記します

そして、関数に持たせようとしている機能を元に関数名を英語で決めて記述します

ダイアログを表示する関数は予め実装されてます。

基本形はalert();で、ダイアログは表示できます

上記の例ではダイアログに文字列を表示させるために、alert関数に文字列を引数(パラメータ)として渡します

文字列はクォーテーション(')で囲むというルールがあります

ここまでは関数の定義をしました

定義した関数は呼び出されるまで実行されません

ボタンがクリックされたときに、先程定義した関数を実行するようにしてあげましょう

ボタンタグにonClickイベントを与えて、先程定義した関数を記述します

<button class='btn' onClick='sayHello()'>ボタン</button>

実装するとこのようになります
